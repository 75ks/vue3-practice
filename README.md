# vue3-practice

## プロジェクト作成
ターミナル起動後、下記を実行
```
// カレントディレクトリを確認(/Users/[Macのユーザー名]になっていること)
pwd

// ローカルにプロジェクトをコピー
git clone https://github.com/75ks/vue3-practice.git
```

## サーバー起動
```
// 作成したプロジェクトディレクトリに移動
cd vue3-practice

// サーバーを起動
npm run serve
```

### サーバー起動確認の為、下記にアクセス
http://localhost:8080/
